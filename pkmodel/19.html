4.2. Контрольно-измерительные приборы Electronics Workbench




Панель контрольно-измерительных приборов в программе включает мультиметр, осциллограф, функциональный генератор и измеритель АЧХ и ФЧХ. К каждому прибору соединяются проводники к узлам схемы. Для работы с прибором необходимо дважды щелкнуть курсором на его иконке. Мультиметр предназначен для измерения напряжения, тока и сопротивления. В нем можно выбрать режим измерения и использовать кнопку dB для измерения напряжения в децибелах. Также доступны вольтметр и амперметр для измерений постоянного или гармонического тока и напряжения.


Выбор АС в осциллографе - для переменных сигналов, по умолчанию используется режим DC. Y position позволяет смещать изображение. Цена деления по вертикали до 5 кВ/дел, горизонталь - в окне Time Base. Expand увеличивает размер окна, есть полоса прокручивания и визирные линии. Соединение с корпусом не обязательно. Функциональный генератор генерирует сигналы с частотой, амплитудой, смещением и duty cycle.


Описание трех приборов: 1) Измеритель АЧХ и ФЧХ предназначен для получения амплитудно-частотных и фазочастотных характеристик четырёхполюсников. 2) Генератор логических сигналов создает на выходных зажимах последовательность 16-разрядных двоичных слов с заданной частотой следования. 3) Анализатор логических сигналов предназначен для индикации двоичных кодов. Для правильного воспроизведения логических сигналов необходимо установить внутреннюю частоту прибора выше частоты генератора логических слов.


Логический преобразователь в программе EWB позволяет получить таблицу состояний комбинационной схемы, преобразовать таблицу истинности в логическую функцию и наоборот, а также преобразовать логическую функцию в схему устройства на логических элементах. В программе предусмотрены автоматические режимы анализа цепи, такие как расчет режима по постоянному току, частотных характеристик и переходных процессов. Также доступны другие, реже используемые виды анализа, такие как спектральный анализ, анализ спектра внутренних шумов, расчет нелинейных искажений и др. Рекомендуется настроить параметры численного интегрирования для повышения точности результатов анализа.


Расчет гармоник в спектре сигнала осуществляется численным интегрированием по формуле ряда Фурье. Вариация параметров полупроводниковых приборов для анализа их вольт-амперных характеристик возможна через пункт Parameter Sweep меню Analysis. Для оценки нестабильности характеристик устройства и их чувствительности к изменениям параметров элементов используются Sensitivity, Worst Case и Monte Carlo. Анализ шумов в малошумящих электронных устройствах проводится через пункт Noise меню Analysis.


Вопросы


1. перечислить и пояснить назначение измерительных приборов аналоговой схемотехники.


- Мультиметр - для измерения напряжения, тока и сопротивления;


- Осциллограф - для визуального отображения формы сигналов и измерения параметров этих сигналов (амплитуда, частота, фаза);


- Генератор сигналов - для создания различных типов сигналов для последующего измерения.


2. перечислить и пояснить назначение измерительных приборов цифровой схемотехники.


- Логический анализатор - для анализа цифровых сигналов на низком уровне;


- Частотомер - для измерения частоты сигнала;


- Протоколирующий анализатор - для анализа протоколов передачи данных.


3. Перечислить и пояснить основные виды анализа схем в программе.


- Симуляция - для проверки работы схемы без реального подключения к оборудованию;


- Анализ переменных - для анализа значений переменных в различных точках схемы;


- Анализ времени - для анализа сигналов и их параметров по времени;


- Анализ частоты - для анализа спектра сигнала и его составляющих частот.
