5.1. Возможности программы Sprint-Layout 5.0




Основным достоинством Sprint-Layout является интуитивно понятный интерфейс и возможность работать с платами размером 300 на 300 мм, двумя слоями для каждой стороны платы, а также добавлять слой паяльной маски, металлизацию, SMD-маску. В пополняемой библиотеке есть наиболее распространенные электронные компоненты. Программа также позволяет экспортировать результаты в форматы Excellon и Gerber, создавать файл HPGL для фрезерного станка. Sprint-Layout подходит для начинающих проектировщиков из-за простоты и логичной структуры. Она доступна на немецком и английском языках, а также есть русифицированный вариант.


1. Масштабирование платы теперь производится колесиком мыши. 2. Все элементы и макросы могут вращаться на произвольный угол. 3. Добавлены 2 внутренних слоя для создания многослойных плат. 4. Паяльную маску теперь можно редактировать. 5. Появилась возможность проверки платы на соответствие установленным требованиям. 6. В качестве единиц можно использовать как миллиметры, так и милы (1/1000 дюйма). 7. Для редактирования элементов платы можно использовать специальные панели свойств. 8. Собственные макросы можно создавать с помощью мастера по созданию фут-принтов. 9. На земляном проводнике можно создавать тепловые барьеры вокруг падов. 10. Можно создавать спирали - например, спиральные печатные индуктивности. 11. Можно самостоятельно определить "горячие клавиши" для большинства операций.


И вот интересный факт: создание печатной платы теперь представляет собой целую отдельную отрасль технического искусства! Печатная плата предназначена для электрического и механического соединения различных электронных компонентов, а сегодня радиолюбителям доступно заводское производство для заказа своих печатных плат. В то же время можно создавать платы и в домашних условиях, используя ЛУТ и дешевый раствор для травления.


- Курсор (Esc): используется для выделения элементов на плате, таких как отверстия или части дорожки. - Масштаб (Z): увеличивает/уменьшает размер рисунка печатной платы, удобен при работе с тонкими дорожками. - Дорожка (L): предназначена для рисования проводящей дорожки. Имеет несколько режимов работы. - Контакт (P): используется для рисования переходных отверстий, позволяет выбирать форму отверстия и задавать радиусы. - SMD-контакт (S): для проектирования ПП с использованием SMD-компонентов, рисует контактные площадки нужных размеров. - Круг / Дуга (R): используются для отрисовки проводника в форме круга или дуги. - Квадрат (Q) / Полигон (F) / Спецформы (N): для создания площадок и областей определенной формы. - Текст (T): для написания текста на плате, можно задать отображение текста зеркально и нормально. - Маска (O): для работы с паяльной маской, позволяет открывать/закрывать маску на контактах и дорожках. - Перемычки (C): виртуальная связь между контактными дорожками, используется для автотрассировки. - Автотрасса (A): простой автотрассировщик, проводит дорожки между контактами. - Тест (X): позволяет подсветить одну конкретную дорожку для проверки правильности разводки. - Измеритель (M): удобен для измерения расстояний на чертеже платы. - Фотовид (V): показывает примерный вид платы после изготовления промышленным способом.


И последнее, о слоях в Sprint Layout: программа позволяет рисовать многослойные платы с семью слоями: два внешних медных слоя (верхний и нижний), два слоя под шелкографию для внешних слоев, два внутренних слоя и один непечатный слой для отрисовки контура платы.


Вопросы


1. Пояснить назначение программы Sprint-Layout 5.0.


Назначение программы Sprint-Layout 5.0: это программа для проектирования печатных плат, обладающая удобным интерфейсом и многочисленными инструментами для создания электронных схем и макетов плат.


2. Перечислить возможности программы.


Возможности программы: создание однослойных и многослойных плат, редактирование паяльной маски и различных элементов, использование специализированных инструментов для рисования дорожек, контактов и текста, автоматическая и вручную установка перемычек, автотрассировка дорожек, а также замеры расстояний на печатной плате.


3. Пояснить основные функции программы при создании проекта платы.


Основные функции программы при создании проекта платы: создание и редактирование элементов, рисование дорожек, выравнивание элементов, установка контактов и отверстий, размещение перемычек, а также измерение расстояний и углов на плате.


4. Пояснить назначение функций Фотовид и Металлизация.


Функция "Фотовид" отображает примерный вид печатной платы после ее изготовления промышленным способом. Функция "Металлизация" позволяет добавлять медные слои на плату, необходимые для формирования дорожек и контактов.
