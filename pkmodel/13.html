3.2. Базовые понятия и определения системы. Основные составляющие системы GPSS World




Вопросы


1. Пояснить основные понятия и определения системы


Основные понятия и определения системы:


- Система - это совокупность взаимодействующих элементов (компонентов), образующих единое целое и имеющих определенную поведенческую модель.


- Компонент - это отдельный элемент системы, обладающий определенными свойствами и функциями.


- Взаимодействие - это связь между компонентами, которая позволяет осуществлять нужные действия для достижения результата.


2. Представление времени в процессе имитации


Представление времени в процессе имитации:


В системе GPSS World время представляется в виде дискретных моментов, называемых транзакциями. Также время можно измерять в единицах, заданных пользователем.


3. Назвать типы объектов, которые можно использовать при создании моделей


Типы объектов, которые можно использовать при создании моделей:


- Терминалы (Facilities) - места, где происходят операции над транзакциями.


- Очереди (Queues) - места, где транзакции ожидают своей очереди на операцию.


- Генераторы (Generators) - объекты, создающие транзакции.


- Транспортные средства (Storages) - объекты, перемещающие транзакции между Facilities.


4. Перечислить операторы языка PLUS


Операторы языка PLUS:


- + (сложение)


- - (вычитание)


- * (умножение)


- / (деление)


- % (остаток от деления)


- ** (возведение в степень)


5. Для чего предназначены строковые процедуры?


Для чего предназначены строковые процедуры:


Строковые процедуры нужны для работы с текстовыми данными, которые используются в модели. Это могут быть имена компонентов, значения параметров и т.д.


6. Какие математические процедуры включает система GPSSW?


Какие математические процедуры включает система GPSSW:


Система GPSS World включает в себя множество математических процедур, например:


- Распределение вероятности по заданному закону.


- Генерация случайных чисел.


- Моделирование статистики на основе имитации.
